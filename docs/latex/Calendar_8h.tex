\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classCalendar}{Calendar}} class, representing active weekdays for calendar in the GTFS dataset. 

This header file declares the \mbox{\hyperlink{classCalendar}{Calendar}} class, which inherits from \mbox{\hyperlink{classGTFSObject}{GTFSObject}}. The class serves as a representation of the GTFS \char`\"{}calendar.\+txt\char`\"{} file, storing information about active days of a calendar.
