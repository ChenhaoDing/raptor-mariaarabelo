\doxysubsection{Detailed Description}
Represents information about a transit stop during a journey. 

This structure holds details about a stop\textquotesingle{}s arrival time, the trip and stop it depends on, and the day of operation. Values are optional to handle cases where a stop is unreachable or is a starting point.
