\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classRaptor}{Raptor}} class for finding Pareto-\/optimal journeys in a transit network. 

This header file declares the \mbox{\hyperlink{classRaptor}{Raptor}} class, which implements the Round-\/\+Based Public Transit Routing algorithm.

The main method involve finding journeys.

The class also contains several private methods for initializing the algorithm, traversing routes, and reconstructing journeys.
