This project implements the Round-\/\+Based Public Transit Routing (RAPTOR) algorithm, enabling efficient public transport route searches based on GTFS data.\hypertarget{md_README_autotoc_md420}{}\doxysubsection{Prerequisites}\label{md_README_autotoc_md420}

\begin{DoxyItemize}
\item C++20 or later
\end{DoxyItemize}
